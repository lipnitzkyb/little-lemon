# Little Lemon Capstone Project

## Overview
This is the final capstone project for the Meta Front-End Developer Professional Certificate.

The application allows users to reserve a table at the fictional Little Lemon restaurant.

## Features
- Responsive design
- Booking form with validation
- Simulated API
- Accessible semantic HTML
- Unit testing with React Testing Library
- Routing with React Router

## Setup Instructions

1. Clone repository:
   git clone https://github.com/your-username/little-lemon.git

2. Install dependencies:
   npm install

3. Run development server:
   npm start

4. Run tests:
   npm test

## Accessibility
- Semantic HTML5 elements
- aria-labels
- role="alert" for error messages
- Accessible form labeling

## Testing
Unit tests are located in:
src/__tests__/

## Deployment
Build for production:
npm run build

Then deploy via Netlify, Vercel, or GitHub Pages.
